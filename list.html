Lists

Topics

  1. Elements
    <div>
    This section can't be grouped into any of the standard elements, but we still want to add style to it. When no other element is suitable, the <div> element is used as an element of last resort. Otherwise, it leads to poor accessibility for readers.
    <dl>
    gives the definition term or basis on what is about to be defined
    <dt>
    is a defintion list term and can be used in question format.
    <dd>
    is a definition list definition and is used as an answer or explanation to the <dt> question.
    <ol>
    A sequenced list; the sequence of the items matters
    <ul>
    A collection of items for which the order is meaningless.
    <li>
    Used to represent a single list item in an ordered list, unordered list, or menu.
  2. Attributes
    3. Reversed
    The reversed attribute is a boolean attribute. If present, it indicates that the list is a descending list (..., 3, 2, 1).
    Start
    The start attribute specifies the start value of the first list item in an ordered list.
    Attribute Value:
    • number: Specifies the start value of the first list item in the ordered list
    Type
    The type attribute specifies the kind of marker to use in the unordered list.
    Attribute Value:
    • disc
    • circle
    • square
    Value
    The value attribute sets the value of a list item. The following list items will increment from that number.
    The value must be a number and can only be used in ordered lists (<ol>).
    Attribute value:
    • number: specifies the value of the list item
